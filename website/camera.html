Camera Stream on Raspberry Pi

The BlueBox is a Raspberry Pi device that stores offline lessons for students without internet access. This enhanced version includes live-streaming running on Flask server, allowing students to connect locally to the device's network on their tablets and participate in real-time activities like experiments or crafts.

❮ ❯

The camera setup for this project is based on EbenKouao's GitHub source code, but significant work was needed to enable live streaming on the user interface. This involved installing and updating packages, adjusting the Pi's OS for compatibility with the camera version used, and improving the Python code to enhance image quality and ensure compatibility with the newly installed packages
